## Layer 2

- <a
  href="https://blog.ethereum.org/2025/12/18/zkevm-security-foundations"
  target="_blank" rel="noopener noreferrer nofollow">Ethereum Foundation
  outlines zkEVM security milestones</a> - The EF cryptography team
  published a roadmap for zkEVM security, shifting focus from
  performance to security. Three key milestones were announced:
  soundcalc integration by February 2026, 100-bit provable security by
  May 2026 (Glamsterdam), and 128-bit provable security by the end of
  2026 (H-star).

## Security

- <a href="https://rekt.news/yearn-rekt4" target="_blank"
  rel="noopener noreferrer nofollow">Yearn Finance exploited for $293k</a> -
  Yearn suffered its second hack in six weeks and fourth exploit
  overall. The attack recycled a 2023 vulnerability to drain funds from
  the protocol.

- <a href="https://rekt.news/aevo-rekt" target="_blank"
  rel="noopener noreferrer nofollow">Aevo (formerly Ribbon Finance) lost
  $2.7 million</a> - A proxy admin vulnerability in old vaults allowed
  an attacker to hijack the oracle, manipulate prices, and drain the
  contract in a single atomic transaction.

## Ecosystem

- <a href="https://blog.ethereum.org/2025/12/16/future-of-state"
  target="_blank" rel="noopener noreferrer nofollow">The Future of
  Ethereum's State</a> - The Stateless Consensus team published a
  proposal addressing Ethereum's state growth challenges. The article
  outlines three approaches: state expiry, state archive, and making it
  easier to hold and serve state, with a focus on reducing state as a
  performance bottleneck.

- <a
  href="https://ethereum-magicians.org/t/2025-upgrade-process-retrospective/27082"
  target="_blank" rel="noopener noreferrer nofollow">2025 upgrade process
  retrospective</a> - The Ethereum community is discussing lessons
  learned from this year's protocol upgrades, with multiple contributors
  sharing insights over the past week.

- <a
  href="https://ethereum-magicians.org/t/erc-8107-ens-trust-registry-for-agent-coordination/27200"
  target="_blank" rel="noopener noreferrer nofollow">ENS Trust Registry
  for Agent Coordination proposed</a> - A new ERC proposal aims to
  create a trust registry using ENS for coordinating AI agents and
  services on Ethereum.
